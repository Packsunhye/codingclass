02. 나의 문제 풀이 

function solution(n){
    var answer = 0;
    var ever = String(n);

    for(var i =0; i < ever.length; i++){
        answer += parseInt(ever[i]);
    }

    return answer;
}

03. 다른 사람의 문제 풀이 

function solution(n){
    var a = (n + '').split('');
    var b = 0;
    for(var i = 0; i < a.length; ++i) {
        b += parseInt(a[i]);
    }
    return b;
}

비슷해보이지만, split로 문자열을 나눈 다음에 갯수를 구하고 parseInt로 정수로 변환하여 더한것 같습니다.
