01. toUpperCase()

문자열을 대문자로 변경해주는 메서드 입니다.

const str1 = "cat Do pUNch";
const currentStr1 = str1.toUpperCase(); // CAT DO PUNCH 


    //toUpperCase()
    const str1 = "javascript";
    const currentStr1 = str1.toUpperCase(); // JAVASCRIPT

02. toLowerCase()

문자열을 소문자로 변경해주는 메서드 입니다.

const str1 = "cat Do pUNch";
const currentStr1 = str1.toUpperCase(); // cat do funch 


    //toLowerCase()
    const str1 = "JAVASCRIPT";
    const currentStr1 = str1.toUpperCase(); // javascript

03. trim()

문자열의 앞/뒤 공백을 제거해주는 메서드 입니다.

const str1 = " Kitty ";
const currentStr1 = str1.trim(); // Kitty 


    //trim()
    const str1 = "        javascript         "
    const currentStr1 = str1.trim(); // javascript

04. trimStart()

문자열의 앞의 공백을 제거해주는 메서드 입니다.

const str1 = " Kitty ";
const currentStr1 = str1.trimStart(); // Kitty_____ 


    //trimStart()
    const str1 = "        javascript         "
    const currentStr1 = str1.trimStart(); // javascript______

05. trimEnd()

문자열의 뒤의 공백을 제거해주는 메서드 입니다.

const str1 = " Kitty ";
const currentStr1 = str1.trimStart(); // _____Kitty 


    //trimEnd()
    const str1 = "        javascript         "
    const currentStr1 = str1.trimEnd(); // ______javascript
