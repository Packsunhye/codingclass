charAt() 란?

search()나 indexOf()와 비슷하나, 숫자로 검색 할 시 검색값이 문자열로 나옵니다.

const str1 = "cat do funch"
const currentStr1 = str1.charAt(); //c
const currentStr2 = str1.charAt('0'); //c
const currentStr3 = str1.charAt("5"); //f
