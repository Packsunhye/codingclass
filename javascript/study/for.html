반복문

반복문은 프로그램에서 필요한 결과 값을 도출하기 위해 실행문의 순서를 반복적으로 실행시키는 문법을 말합니다. 반복문에는 while문 , do while문, for문을 가장많이 사용합니다.

while문

while ( 조건식){
    실행문;
    중가감식;
}

do while문

do {
    실행문
    증가감식;
} while(조건식)

for문

for ( 초기값; 조건식; 증강값 ){
    실행문
}

구구단 출력하기

중첩 for문을 이용해 구구단을 출력할 수 있습니다.

   for(let i = 2; i<=9; i++){
        for(let j =1; j<=9; j++){
            let sum = i * j ;
            document.write(i + " * " + j + " = " + sum);
            document.write("<br>");
        }
    }
